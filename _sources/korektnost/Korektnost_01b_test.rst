.. mchoice:: korektnost_q03
    :answer_a: Програм ће сваки пут бити заустављен у току извршавања.
    :answer_b: Програм може да буде заустављен у току извршавања, што зависи од дужине низа.
    :answer_c: Програм ће завршити са радом, али низ b најчешће неће бити исправно попуњен
    :answer_d: Програм ће радити исправно за сваки непразан низ a
    :correct: c
    :feedback_a: Не.
    :feedback_b: Не.
    :feedback_c: Тачно!
    :feedback_d: Не.
    
    Дат је непразан низ *a*. Потребно је да се формира низ *b* исте дужине, у коме за сваки дозвољени индекс *i* вредност *b[i]* треба да буде једнака збиру елемената од почетка низа *a*, закључно са позицијом *i*. Следећи код би требало да обавља тај задатак.
    
    .. code::

        int n = a.Length;
        int[] b = new int[n];
        b[0] = a[0];
        for (int i = 1; i < n - 1; i++)
            b[i + 1] = b[i] + a[i + 1];

.. mchoice:: korektnost_q05
    :multiple_answers:
    :answer_a: Тест А може да се изостави јер је сувише једноставан
    :answer_b: Тест В може да се изостави јер проверава исто што и Б
    :answer_c: Свака неисправна функција на бар једном од ових тестова не даје исправан резултат
    :correct: b
    
    Нека функција треба да испише вредности своја три параметра, од најмање до највеће. У тој функцији се не користе никакве константе. Јован намерава да тестира функцију за следеће вредности параметара:
    
    тест А: (1, 2, 3), тест Б: (15, 23, 18), тест В: (2, 5, 3), тест Г: (4, 2, 7),

    тест Д: (17, 30, 12), тест Ђ: (15, 12, 18), тест Е: (52, 45, 31)
    
    Означи тачне исказе о Јовановим тест-примерима.

    .. code::

        int s = 0;
        for (int i = 1; i < n++; i++)
            s = s + i;
        Console.WriteLine(s);
